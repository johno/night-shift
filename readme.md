# night-shift [![js-standard-style](https://img.shields.io/badge/code%20style-standard-brightgreen.svg?style=flat)](https://github.com/feross/standard)

Toggle OS X Night Shift.

#### installation

```sh
npm i -S night-shift
```

#### usage 

```javascript
const nightShift = require('night-shift')

nightShift()
```

#### cli

```sh
npm i -g night-shift
night-shift -h
```
